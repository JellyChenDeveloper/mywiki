# 搭建个人知识管理系统docsify

[官网地址](https://docsify.js.org/#/)

## 开始

1. 安装 `docsify-cli` 工具即可。

    ```shell
    cnpm i docsify-cli -g
    ```

2. 初始化目录

    ```shell
    cd wiki
    docsify init .
    ```

3. 本地预览网站

    运行一个本地服务器通过 docsify serve 可以方便的预览效果，而且提供 LiveReload 功能，可以让实时的预览。默认访问 [http://localhost:3000](http://localhost:3000) 。

    ```shell
    docsify serve .
    ```
